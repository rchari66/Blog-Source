### Web Server 
* Servers only web pages. Could be static or Dynamic pages
    * Static Pages
        * HTML
        * CSS
        * JavaScript(Interactively)
    * Dynamic Pages
        * JSP, Servlet (JAVA + HTML + CSS + JavaScript)
        * PHP (PHP + HTML + CSS + JavaScript)
        * ASP.NET (C# + HTML + CSS + JavaScript)
